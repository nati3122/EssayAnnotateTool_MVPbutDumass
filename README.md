# Essay Annotate Tool

An intelligent essay annotation system that combines OCR and NLP technologies to analyze and annotate academic essays.

## Features

- **OCR Engine**: Extracts text from essay documents (PDF/image formats)
- **NLP Engine**: Performs natural language processing for linguistic analysis
- **Annotation System**: Automated annotation and feedback generation
- **PDF Support**: Process and annotate PDF documents

## Project Structure

- `annotator.py` - Main annotation logic and workflow
- `nlp_engine.py` - NLP processing and analysis
- `ocr_engine.py` - OCR text extraction
- `test_main.py` - Testing and validation

## Installation

Clone the repository:
```bash
git clone https://github.com/nati3122/EssayAnnotateTool_MVPbutDumass.git
cd EssayAnnotateTool_MVPbutDumass
```

Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the annotation tool:
```bash
python annotator.py
```

Run tests:
```bash
python test_main.py
```

## Requirements

- Python 3.x
- NLP and OCR dependencies (see requirements.txt)
